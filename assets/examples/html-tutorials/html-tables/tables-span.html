It can get really tricky to span cells as you need to get rid of one td tag in the desired location(s) of the span. If you are spanning a cell into two columns, you would need to get rid of 1 td tag in the same row as the cell you are spanning. If you are spanning a cell into 2 rows, you would need to get rid of 1 td tag in the row that you are spanning in to. If you don't do this, your table will look messed up.
